O nosso software de reconhecimento facial pode ser facilmente integrado a
sua empresa. Nunca foi tão simples realizar o gerenciamento de entrada e saída de pessoas.

Qual o impacto
do LeopardTech?

Trazendo o reconhecimento facial para a
educação e implementação de inovações
tecnologicas de segurança nas empresas.

Etapas do projeto


  • Criar o sistema de cadastro para os alunos, com a
    matricula e foto do aluno(SQL);

  • Criar o sistema de reconhecimento facial (Python);

  • Conforme o aluno entra pela catraca, realizar uma
    consulta no banco de dados para ver a qual turma ele pertence
    e marcar presença no NSA ou criar uma lista (possível também
    a criação de dashboards com o desempenho de cada aluno ligado a sua presença);

  • Conectar o sistema em uma catraca, juntamente com o monitor e webcam;

  • Cadastrar alunos para testes.
